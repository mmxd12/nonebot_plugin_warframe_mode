# nonebot-plugin-warframe_mode

基于NoneBot2的WM市场查询的插件


# 安装

- 脚手架安装

    ```shell
    pip install nonebot-plugin-warframe_mode
    ```

- NoneBot2 插件商店安装

    ```shell
    nb plugin install nonebot-plugin-warframe_mode
    ```

# 使用

- 前缀：`/`
- 指令菜单：`救命 菜单`
  ```text
  注意：所有指令需要搭配前缀使用！
  指令参数：
  ==================
  wm <搜索词>
  rm <搜索词> 
  wiki <搜索词>
  翻译 <搜索词>
  打折|小小黑|特价
  新闻| 事件 |警报
  突击| 裂缝 |入侵
  奸商| 特价 |地球
  电波| 仲裁 |扎里曼
  地球赏金|金星赏金|火卫二赏金
  火卫二平原|金星平原|地球平原
  ===================
  当前版本0.0.4
  ```
  
# 注意
- python版本需要3.9以上，内置新语法！
